## Recapitularea Noţiunilor `~` Curs PHP 2021 - 2022
---
##### Bloc 
- Una sau mai multe linii de programare, care constituie o unitate logică.
##### Browser
 - Deseori, noţiunile de motor de căutare şi browser se confundă. Un motor de căutare poate fi o aplicaţie web care afişează informaţiile colectate de pe internet la cererea unui utilizator (Google, Yahoo...). Browserul este un program folosit de un client pentru a comunica cu internetul (Edge, Firefox, Opera, Chrome...).
##### Browser web
 - Program care comunică cu serverul web şi textul marcat cu ajutorul semnului HTML, care se prezintă în forma simplă pentru vizualizarea conţinutului paginii de către utilizator.
##### Buclă
 - Parte din codul de programare care se execută de câteva ori.
##### Caractere speciale
 - Există anumite caractere care în programare se numesc speciale (de exemplu, #,$,%,/...). De obicei, aceste caractere au propria valoare în sintaxa limbajului de programare şi cel mai des este interzisă utilizarea lor în partea neîncorporată a codului.
##### Clasă 
- Model care defineşte anumite caracteristici şi comportamente, pe baza căruia pot fi create obiecte individuale.
##### Client, Server
 - Când este vorba de comunicarea pe internet, întotdeauna trebuie să existe două părţi, şi anume: clientul, care a activat serverul cu o cerere, şi serverul, care procesează cererea respectivă. Serverul este descris deja şi este destul de clar, în timp ce clientul reprezintă o platformă care trimite cereri către server şi tot de pe acesta preia răspunsurile. Deci, clientul nu este o persoană care navighează pe internet cu ajutorul unui browser, ci acesta este tocmai browserul în sine (Internet Explorer, Mozilla Firefox). Persoana care foloseşte aplicaţia client se numeşte utilizator.
##### Cod HTML al paginii
 - Fiecare pagină web este alcătuită exclusiv din codul HTML. Acesta este limbajul cu a cărui interpretare browserul web emite conţinutul. Acest cod este întotdeauna public şi poate fi accesat din orice browser web.
##### Cod sursă
 - Acesta este programul în forma generală, înainte de a trece prin orice procesare.
##### Concatenare
 - Adăugarea stringului la stringul existent – legarea stringurilor.
##### Constructor
 - Acest termen este strâns legat de instanţierea claselor. Constructorul este, de fapt, funcţia ai cărei parametri se preiau direct din parametrii pentru crearea clasei. De exemplu: $myClass = new myClass($myParameter); Partea din cod (funcție-metodă) care, în cadrul unei clase, gestionează acest parametru se numeşte constructor.
##### DSN (Data Source Name)
 - Informaţie despre o bază de date, despre locaţia acesteia, modul de logare, sistemul de administrare şi altele, folosită în cadrul ODBC.
##### Expresie
 - Combinaţie de variabile, literali şi operatori care returnează o anumită valoare.
##### Fişier php.ini
 - Fişier de configurare cu ajutorul căruia se pot controla unele setări ale interpretorului PHP.
##### Formatarea textului
 - Reprezentarea textului într-o anumită formă.
##### Formular
 - Set de obiecte (controale) de pe o pagină, care permit interacţiunea utilizatorilor. De obicei, este destinat pentru colectarea datelor de la utilizatori.
##### Funcţie
 - În programare, funcţia este un subprogram independent, care efectuează o anumită activitate, interacţionând cu mediul prin intermediul parametrilor de intrare şi ieşire. Dacă ar exista funcţia care adună două numere, aceasta ar avea cele două numere ca parametri de intrare. Funcţiile reprezintă noţiunea esenţială în programare şi despre ele vom discuta mai mult în următoarele lecţii.
##### Funcţie Anonymous
 - Funcţie care se iniţializează în momentul executării, nu are identificare şi nu poate fi utilizată ulterior. De obicei, se atribuie evenimentelor.
##### HTML
 - Modalitate de marcare a textului. În cadrul lui HTML, există un set de taguri definite în avans pentru marcarea textului, pe care browserele pot să le înţeleagă şi pe baza cărora afişează textul utilizatorului. Cel mai des se foloseşte pentru reprezentarea paginilor web.
##### Încapsulare
 - Când este vorba de programare, încapsularea presupune o arhitectură ascunsă a unui sistem cu care comunică utilizatorul prin intermediul unei anumite interfeţe. Sau, mai simplu: pentru programul care adună două numere şi despre el ştim doar că poate să accepte două numere şi să returneze rezultatul, fără a şti cum funcţionează de fapt şi ce conţine, se poate spune că este încapsulat.
##### Instrucţiune de ramificare
 - Instrucţiune de programare, care permite executarea unei părţi, sau a celeilalte, a codului de programare, în funcţie de exactitatea unei condiţii.
##### Interpolare
 - Proces de schimb de variabilă cu conţinutul ei în cadrul unui string.
##### Intranet
 - Intranetul este o reţea locală care funcţionează prin intermediul tehnologiilor de internet. Serverul web local, site local, aplicaţii web locale, adrese IP locale...
##### Iteraţie
 - Trecere secvenţială printr-o listă de date. De obicei, trecerea prin elementele şirului dintr-o buclă se numeşte iteraţie.
##### Moştenirea clasei
 - Alocarea caracteristicilor şi a metodelor, pe care le deține o clasă, la o altă clasă care o moşteneşte.
##### Obiect
 - Un exemplar individual de clasă care deține toate caracteristicile şi comportamentele pe care le defineşte această clasă.
##### ODBC (Open Database Connectivity)
 - Set standardizat de funcţii pentru comunicarea cu diferite sisteme pentru gestionarea bazelor, dezvoltat de Microsoft.
##### Parsare
 - Împărţirea conţinutului documentului în elemente care pot fi identificate.
##### Port TCPIP
 - Număr care identifică o conexiune unică a calculatorului local cu cel de la distanţă.
##### Script PHP
 - Secţiunea codului PHP în cadrul unei pagini web.
##### Server web
 - Program care permite acceptarea cererii HTTP prin intermediul internetului şi emiterea răspunsurilor corespunzătoare la aceste cereri. De asemenea, server web este şi calculatorul pe care se execută acest program.
##### Sesiune
 - Timpul pe care utilizatorul îl petrece pe un site, adică timpul de la intrarea utilizatorului pe site până la părăsirea acestuia.
##### Şir
 - Structură de date pentru salvarea setului sau a secvenţei de valoare.
##### Şir asociativ
 - Şir care este indexat cu valori de tip string, în loc să fie indexat cu valori numerice.
##### SQL
 (Structured Query Language) - Limbaj standardizat pentru setarea interogării asupra bazelor.
##### String
 - Şir de caractere.
##### Tag
 - Semn care în limbajul HTML specifică modul în care va fi afişat un text sau ce va fi inserat într-o pagină web. În majoritatea cazurilor, există taguri de deschidere şi de închidere, de exemplu, acesta este un text. Atunci când un parser HTML întâlneşte un astfel de tag (iar semnele tagului, în acest caz, sunt and ), conţinutul dintre ele emite litere aldine/îngroşate. Pe lângă conţinut, tagurile pot avea şi propriile lor atribute. conţinut Tagurile trebuie să fie întotdeauna închise, fie că se închid cu o etichetă specială, fie că se închid în cadrul propriei iniţializări (aceste taguri se numesc taguri cu autoînchidere).
##### Tipuri simple şi complexe de date
 - Tipul simplu de date este, de obicei, în raport de 1:1 cu memoria, respectiv o locaţie de memorie poate să conţină valoarea unui tip. De asemenea, de obicei, este nevoie de o instrucţiune a procesorului pentru ca acest tip să fie preluat din memorie, în timp ce tipurile complexe au fixată în memorie doar adresa iniţială a întregului obiect.
##### URL (Uniform Resource Locator)
 - Informaţii despre serverul web şi calea pe care se află o pagină web.
##### Valoare iniţială
 - Valoare care se atribuie unei variabile sau unui obiect cu ocazia creării sale.
##### Variabilă
 - Identificatorul adresei în memorie, unde este depozitată o anumită valoare.
